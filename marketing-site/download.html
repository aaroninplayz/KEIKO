code Alternative Installation & Setup Methods

Choose the deployment method that fits your environment.

1

GitHub Desktop Installer

Best for non-technical users and recruiters who want a ready-to-run desktop application.

  1. Visit the official GitHub Release page.
  2. Download Keiko-Setup-x64.exe.
  3. Run the installer and launch KEIKO Desktop.
2

Python Local Engine

Cross-platform option for Windows, macOS, and Linux developers to run from source code.

# Clone & install dependencies
git clone https://github.com/keiko-ai/keiko-platform.git
cd keiko-platform/local-app
pip install -r requirements.txt
# Start local server
python main.py
3

Docker Container

Ideal for containerized deployments, cloud servers, or isolated sandbox evaluation.

# Pull official container
docker pull keikoai/keiko-engine:latest
# Run container on port 8000
docker run -d -p 8000:8000 keikoai/keiko-engine
